# מדריך ביורוקרטיה ישראלית

הביורוקרטיה הישראלית ידועה כמורכבת. בין משרדי ממשלה, ביטוח לאומי, שירותים עירוניים וים של טפסים, גם משימות פשוטות יכולות לקחת שעות של מחקר והמתנה. החבילה הזו מאגדת חמישה סקילי AI שעוזרים לכם לנווט במערכת, להבין את הזכויות שלכם ולסיים דברים מהר יותר.

## למי זה מתאים?

- **עולים חדשים** שלומדים לנווט במערכות הממשלתיות הישראליות
- **אזרחים** שמתמודדים עם ביטוח לאומי, משרדים עירוניים או טפסי ממשלה
- **בעלי עסקים קטנים** שמטפלים באישורים, רישיונות ודרישות רגולטוריות
- **כל מי** שאי פעם נשלח למשרד הממשלתי הלא נכון

## מה מקבלים

1. **מפענח ביורוקרטיה ישראלית** -- תארו את המצב שלכם וקבלו מפת דרכים ברורה: איזה משרד, אילו מסמכים, מה לצפות
2. **ביטוח לאומי ישראלי** -- הבנת גמלאות ביטוח לאומי, זכאות ואיך להגיש
3. **אוטומציה לטפסי ממשלה ישראליים** -- עזרה במילוי טפסי ממשלה עם הדרכה שדה-אחרי-שדה ואימות
4. **ממשק נתוני ממשלה ישראלי** -- גישה לנתונים ושירותים ממשלתיים פתוחים באופן תכנותי
5. **לוחם קנסות ישראלי** -- ניווט בתהליך הערעור על דוחות תנועה וחנייה
