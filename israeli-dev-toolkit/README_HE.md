# ערכת כלים למפתחים ישראליים

בניית תוכנה לשוק הישראלי מגיעה עם אתגרים ייחודיים: פריסת RTL, טיפול בטקסט עברי, אימות תעודת זהות, פורמט מספרי טלפון וקווירקים של לוקליזציה שרוב הפריימוורקים הבינלאומיים לא מטפלים בהם. החבילה הזו נותנת לכם חמישה סקילי AI שפותרים את משימות הפיתוח הישראליות הנפוצות ביותר.

## למי זה מתאים?

- **מפתחים** שבונים אפליקציות או אתרים למשתמשים ישראליים
- **צוותים** שמוסיפים תמיכה בעברית/RTL למוצרים קיימים
- **סטארטאפים** שמשיקים בשוק הישראלי
- **מפתחים פרילנסרים** שעובדים עם לקוחות ישראליים

## מה מקבלים

1. **בינאום לעברית** -- הגדרת בינאום מלאה לעברית, כולל תרגומים, פורמט תאריכים/מספרים והגדרת locale
2. **שיטות מומלצות ל-RTL בעברית** -- תכונות CSS לוגיות, טקסט דו-כיווני ודפוסי פיתוח RTL-first
3. **מאמת תעודת זהות ישראלית** -- אימות מספרי תעודת זהות עם אלגוריתם ספרת הביקורת הנכון
4. **מפרמט טלפון ישראלי** -- נרמול ופורמט של כל סוגי מספרי הטלפון הישראליים
5. **פריסט Tailwind לעברית** -- הגדרת Tailwind CSS מוכנה עם גופנים עבריים, כלי RTL ודפוסי עיצוב ישראליים
